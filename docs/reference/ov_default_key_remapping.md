# Default keyboard remapping for ov_scouter

These key remappings only apply to input into the scouting bar (when the
scouter has been started with `scout_mode = "type"`).

## Usage

``` r
ov_default_key_remapping(scout_mode = "click")
```

## Arguments

- scout_mode:

  string: either "click" for the guided point-and-click scouting
  interface, or "type" for the typing-based interface. Currently
  remapping has no effect with "click"

## Value

A named list

# Default setter calls table (where the setter has instructed the middle hitter to run)

Default setter calls table (where the setter has instructed the middle
hitter to run)

## Usage

``` r
ov_default_setter_calls_table(data_type = "indoor")
```

## Arguments

- data_type:

  string: "indoor", "beach". The returned table will be empty for
  "beach"

## Value

A tibble
